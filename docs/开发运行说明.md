# 开发运行说明

## 1. 后端服务

```powershell
cd backend
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
copy .env.example .env
uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
```

如果希望真实模型优先使用 GPU，建议补装 CUDA 版 PyTorch：

```powershell
python -m pip install --index-url https://download.pytorch.org/whl/cu126 torch==2.11.0+cu126 torchvision==0.26.0+cu126
```

访问地址：

- API 根地址：`http://127.0.0.1:8000`
- Swagger：`http://127.0.0.1:8000/docs`
- 结果静态资源：`http://127.0.0.1:8000/storage/...`

## 2. Web 大屏

```powershell
cd frontend\uav-inspection-web
copy .env.example .env
npm install
npm run dev
```

默认地址：`http://127.0.0.1:5173`

## 3. WPF 桌面端

```powershell
dotnet build frontend\UAVInspectionDesktop\UAVInspectionDesktop.csproj
dotnet run --project frontend\UAVInspectionDesktop\UAVInspectionDesktop.csproj
```

## 4. 分析能力说明

- 图片分析：上传单张图片，返回热力分割图、二值掩码图和面积统计。
- 视频分析：上传视频后进行抽帧处理，历史结果可在 Web 大屏回看。
- 实时流分析：提交流地址后进行采样分析；开发阶段可使用 `demo://field-inspection` 作为演示流。

## 5. 替换真实模型

当前版本默认使用 mock 分割器，真实模型接入时只需要替换后端 `ModelRunner` 实现，并保持输出：

- `probability_map`
- `binary_mask`
- `confidence_summary`

当前系统也支持直接加载 `best.ckpt` 形式的 PyTorch checkpoint，默认按以下配置运行：

```env
MODEL_BACKEND=ckpt
MODEL_TYPE=deeplabv3plus_resnet34
MODEL_PATH=C:/path/to/best.ckpt
MODEL_DEVICE=auto
MODEL_CLASS_INDEX_CROP=1
MODEL_CLASS_INDEX_WEED=2
```

说明：

- `MODEL_DEVICE=auto` 时，CUDA 可用则优先走 `cuda:0`，否则自动回退 `cpu`
- 当前类别映射固定为：`背景 / 作物 / 杂草`
- 如果你的权重训练时把烟株和杂草顺序反了，只需要交换 `MODEL_CLASS_INDEX_CROP` 和 `MODEL_CLASS_INDEX_WEED`
- 热力图和面积统计仅针对“杂草类”生成

## 1. 环境要求

- Windows
- .NET SDK 9
- Python 3.12 或兼容版本
- Git

## 2. 后端启动

```powershell
cd E:\Uav-Field-Inspection-System\backend
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
uvicorn app.main:app --reload
```

访问：

- Swagger：`http://localhost:8000/docs`
- 系统状态：`http://localhost:8000/api/system/status`
- WebSocket：`ws://localhost:8000/ws/realtime`

## 3. 前端启动

```powershell
cd E:\Uav-Field-Inspection-System
dotnet build frontend\UAVInspectionDesktop\UAVInspectionDesktop.csproj
dotnet run --project frontend\UAVInspectionDesktop\UAVInspectionDesktop.csproj
```

前端默认连接 `http://localhost:8000`。

## 4. 配置说明

复制 `backend/.env.example` 为 `backend/.env` 后可修改：

- `DATABASE_URL`：当前数据库连接。
- `MYSQL_DATABASE_URL`：预留 MySQL 连接。
- `RTMP_URL`：RTMP 视频流地址。
- `HLS_URL`：HLS 视频流地址。
- `API_REFRESH_INTERVAL_SECONDS`：WebSocket 推送间隔。
