# 接口文档

默认服务地址：`http://localhost:8000`

## GET /api/system/status

返回系统状态、服务器时间和健康状态。

```json
{
  "system_name": "大田无人机巡检监控系统",
  "status": "系统正常",
  "running": true,
  "server_time": "2026-04-26T15:40:00Z",
  "health": "healthy"
}
```

## GET /api/video/status

返回视频链路状态。

```json
{
  "rtmp_status": "connected",
  "hls_status": "available",
  "fps": 29.8,
  "latency_ms": 86,
  "resolution": "1920x1080"
}
```

## GET /api/detection/statistics

返回检测统计。

```json
{
  "total_count": 1286,
  "current_minute_count": 17,
  "risk_index": 63.5,
  "recorded_at": "2026-04-26T15:40:00Z"
}
```

## GET /api/alerts

返回告警事件列表，支持 `limit` 查询参数。

```json
[
  {
    "id": 1,
    "occurred_at": "2026-04-26T15:32:00Z",
    "alert_type": "病害疑似",
    "content": "3号烟田东侧发现疑似赤星病斑块",
    "confidence": 0.91,
    "status": "待复核"
  }
]
```

## GET /api/config

返回系统配置和数据库连接策略。

```json
{
  "app_name": "大田无人机巡检监控系统",
  "environment": "development",
  "database_type": "SQLite",
  "database_url": "sqlite:///./uav_inspection.db",
  "mysql_database_url": "mysql+pymysql://uav_user:uav_password@127.0.0.1:3306/uav_inspection",
  "rtmp_url": "rtmp://127.0.0.1/live/uav-field",
  "hls_url": "http://127.0.0.1:8080/hls/uav-field.m3u8",
  "refresh_interval_seconds": 2
}
```

## WS /ws/realtime

每 2 秒推送一次聚合数据。

```json
{
  "event": "dashboard.realtime",
  "emitted_at": "2026-04-26T15:40:00Z",
  "system": {},
  "video": {},
  "detection": {},
  "alerts": [],
  "config": {}
}
```
