# 接口文档

默认服务地址：`http://localhost:8000`

## GET /api/system/status

返回系统状态、服务器时间和健康状态。

```json
{
  "system_name": "大田无人机巡检监控系统",
  "status": "系统正常",
  "running": true,
  "server_time": "2026-04-26T15:40:00Z",
  "health": "healthy"
}
```

## GET /api/video/status

返回视频链路状态。

```json
{
  "rtmp_status": "connected",
  "hls_status": "available",
  "fps": 29.8,
  "latency_ms": 86,
  "resolution": "1920x1080"
}
```

## GET /api/detection/statistics

返回检测统计。

```json
{
  "total_count": 1286,
  "current_minute_count": 17,
  "risk_index": 63.5,
  "recorded_at": "2026-04-26T15:40:00Z"
}
```

## GET /api/alerts

返回告警事件列表，支持 `limit` 查询参数。

```json
[
  {
    "id": 1,
    "occurred_at": "2026-04-26T15:32:00Z",
    "alert_type": "病害疑似",
    "content": "3号烟田东侧发现疑似赤星病斑块",
    "confidence": 0.91,
    "status": "待复核"
  }
]
```

## GET /api/config

返回系统配置和数据库连接策略。

```json
{
  "app_name": "大田无人机巡检监控系统",
  "environment": "development",
  "database_type": "SQLite",
  "database_url": "sqlite:///./uav_inspection.db",
  "mysql_database_url": "mysql+pymysql://uav_user:uav_password@127.0.0.1:3306/uav_inspection",
  "rtmp_url": "rtmp://127.0.0.1/live/uav-field",
  "hls_url": "http://127.0.0.1:8080/hls/uav-field.m3u8",
  "refresh_interval_seconds": 2
}
```

## WS /ws/realtime

每 2 秒推送一次聚合数据。

```json
{
  "event": "dashboard.realtime",
  "emitted_at": "2026-04-26T15:40:00Z",
  "system": {},
  "video": {},
  "detection": {},
  "alerts": [],
  "config": {},
  "latest_analysis": {}
}
```

## POST /api/analysis/images

上传单张图片并同步返回热力分割结果。

返回字段包含：

- `job`：任务摘要
- `result.source_image_path`
- `result.heatmap_image_path`
- `result.mask_image_path`
- `result.weed_coverage_ratio`
- `result.weed_pixel_area`
- `result.estimated_plant_count`

## POST /api/analysis/videos

上传视频并创建异步分析任务。返回任务详情，之后通过 `GET /api/analysis/jobs/{job_id}` 或 `WS /ws/analysis/{job_id}` 查看进度和结果。

## POST /api/analysis/streams

提交流地址并创建实时流分析任务。

```json
{
  "source_url": "demo://field-inspection"
}
```

## GET /api/analysis/jobs

分页查询分析历史记录，支持：

- `page`
- `page_size`
- `source_type=image|video|stream`
- `status=pending|running|completed|failed|stopped`

## GET /api/analysis/jobs/{job_id}

返回任务详情，包括：

- `job`
- `latest_result`
- `frames`

## GET /api/analysis/jobs/{job_id}/results

返回与任务详情相同的结果视图，便于前端按“结果页”直接读取。

## POST /api/analysis/jobs/{job_id}/stop

停止视频或实时流任务。

## WS /ws/analysis/{job_id}

推送指定任务的实时更新：

```json
{
  "event": "analysis.job.update",
  "emitted_at": "2026-04-28T10:30:00Z",
  "job": {},
  "latest_result": {},
  "latest_frame": {}
}
```
